## Developing

**Software stack**
``` sh
- Javascript
- React
- Express
- Node

```
**Installing codebase and packages**
```shell
git clone https://github.com/jchan58/GTSP.git
cd JHU-Nimbus/code/frontend
npm install
cd JHU-Nimbus/code/backend
npm install
```

**Local development**
```shell
cd JHU-Nimbus/code/frontend
npm run start
cd JHU-Nimbus/code/backend
npm run dev
```
